# Быстрый запуск тестирования диагностики на чМОСи

1. Скачиваем архив, закидываем распакованный на флешку.
2. Пихаем флешку в комп с чМОСью.
3. Заходим в проводнике в распакованную папку.

Пункты 4 и 5 делаем один раз если пароль рута на тачках одинаковый.

4. В файле `start.sh` указываем пароль рута.
5. В файле `config.py` указываем данные по своей школе полученные от рука (из внешней оценки из ЭЖД).
6. ПКМ в проводнике в папке проекта -> Открыть терминал в этой папке.
7. Перетаскиваем файл `start.sh` в окно терминала -> Enter. 
8. Идем к следующему компу и делаем то же самое.
